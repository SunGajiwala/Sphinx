Retraining the model
====================

To retrain the model on a dataset, use the following steps:

#. Clone the repository to your local machine:

    **git clone https://github.com/DSCI-Admissions-project/admsm_ml_engineering**

#. Install the required dependencies

    **pip install -r requirements.txt**

#. Navigate to the *model* directory

    **cd model**

#. Run the jupyter notebook named *train_model.ipynb*

#. To view new model metadata navigate to *saved* directory in *model*

    **cd model**
